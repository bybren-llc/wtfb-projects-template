## Quick Start

This skill activates automatically when you:

- Introduce characters
- Write dialogue blocks
- Format character names
- Handle dual dialogue
- Use character extensions (V.O., O.S., CONT'D)

## What This Skill Does

Provides Fountain format character and dialogue formatting rules covering
character name formatting, extensions (V.O., O.S., CONT'D), dual dialogue,
parentheticals, and proper dialogue block structure.
